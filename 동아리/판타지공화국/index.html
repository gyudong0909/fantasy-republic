개발 작업 순서

이 순서대로 만들면 됩니다

PHASE 1
프롤로그 시네마틱
대본 텍스트 타이핑 연출 + 배경 전환 + BGM
→ 이것만 완성해도 발표/시연 가능
PHASE 2
정치 시스템
후보 카드 UI + 당 가입 + 투표 + 결과 연출
→ 선거 진행 가능
PHASE 3
금융 관리 (은행)
플레이어 등록 + 자산 관리 + 대출/이체 + 운영자 대시보드
→ 현금 거래 추적 가능 (주식보다 먼저)
PHASE 4
경제 시스템 (주식)
회사/주가 데이터 + 매수/매도 + 정보 카드 + 차트
→ 2라운드 주식 게임 진행 가능
PHASE 5
도박 시스템
경마 + 도박 + 복권
→ 3~5라운드 확장 콘텐츠
PHASE 6
전체 연결 + 마무리
라운드 루프 + 순위표 + 최종 결산
→ 풀 게임 완성
